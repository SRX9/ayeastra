Board Mode is the print-optimized executive view — the screen you open when presenting to leadership or exporting the quarterly board pack.

## What it shows
The board pack rides the same append-only briefing system as weekly briefings: an executive summary, the quarter's material moves per competitor, validated patterns and their track records, and outcomes from tracked actions. Everything is evidenced and stamped with the context version that produced it.

## Print and export
Board Mode strips the app chrome (dock, menu bar) and flips to a paper-white print stylesheet, so printing or saving to PDF produces a clean document directly from the browser.
